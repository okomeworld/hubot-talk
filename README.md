# hubot-talk

hubotがお話するようになります

See [`src/talk.coffee`](src/talk.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-talk --save`

Then add **hubot-talk** to your `external-scripts.json`:

```json
[
  "hubot-talk"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
